## Requisitos

* [Docker](https://docs.docker.com/get-docker/)
* [Gatling](https://gatling.io/open-source/), a performance testing tool
* Make (optional)

## Stack

* 2 Bash apps 
* 1 PostgreSQL (com Stored Functions)
* 1 NGINX
 
## Inicializando a aplicação

```bash
$ docker compose up 
```
