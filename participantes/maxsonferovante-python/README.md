## Maxson Almeida
Projeto feito em Python para a rinha de backend 2024 q1.
* Python 3.11.
* Nenhum framework foi utilizado, criei um handler wsgi puro que roda com o fastwsgi.
* No banco psycopg (c extension) e psycopg_pool
* Postgresql 16.
* Nginx.
