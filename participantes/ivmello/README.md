# Submissão para Rinha de Backend, Segunda Edição: 2024/Q1 - Controle de Concorrência

## ivmello
Submissão feita com:
- `nginx` como load balancer
- `postgres` como banco de dados
- `golang` para api com as libs `fiber`, `pgx` e `pgxpool`
