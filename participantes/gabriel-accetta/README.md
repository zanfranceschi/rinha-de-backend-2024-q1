## Stack

- `NodeJS` Plataforma de Aplicação
- `Typescript` Linguagem de Programação
- `Fastify` Web Framework
- `Prisma` ORM
- `Postgres` Banco de Dados
- `NGINX` Load Balancer

## Run

```bash
    docker compose up
```
