###### Detalhes de execução e montagem do projeto:

- Dockerizar o projeto:

```shell
docker build -t lipeferreira1609/dev_bank_with_dotnet_mn_api:X.Y .
```

- Montar um volume com docker-compose:

```shell
docker-compose up -d
```

- Desmontar um volume com docker-compose:

```shell
docker-compose down
```
