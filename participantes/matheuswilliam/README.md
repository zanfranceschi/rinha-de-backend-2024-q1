<h1 align="center">Rinha de Backend - 2024 Q1</h1>

Esse projeto foi feito usando:
- C
- Postgresql
- NGINX

## Instrução
Para rodar é nescessario ter o Docker instalado, e depois rodar o comando abaixo no terminal:

```sh
$ docker-compose up
```
