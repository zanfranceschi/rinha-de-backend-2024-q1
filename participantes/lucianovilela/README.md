# Rinha de backend 2024 q1

## Descrição

Este é um projeto de exemplo que usa Docker, Node.js, NGINX e PostgreSQL.

## Ferramentas utilizadas

Antes de começar, você precisa ter instalado em sua máquina as seguintes ferramentas:
- nodejs
- ngnix
- postgresql
