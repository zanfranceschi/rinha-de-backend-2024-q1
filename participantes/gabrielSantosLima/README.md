![](./docs/fastmoney.png)

## Gabriel Dos Santos Lima

Submissão feita com:

- `nginx` como load balancer
- `postgres` como banco de dados
- `Node.js` para api com `express.js (framework)` e `typescript`. Também utilizando a biblioteca `knex.js` para conexão com o banco.
