## :information_source: How to run?

##### Pre-requisites:

  - Gatling
  - Docker
  - docker-compose


```bash
$ git clone git@github.com:kleytonsolinho/rinha-de-backend-2024-q1-golang.git
```

```bash
$ cd rinha-de-backend-2024-q1
```

```bash
$ docker-compose up -d
```

```bash
$ ./executar-teste-local.sh
```
