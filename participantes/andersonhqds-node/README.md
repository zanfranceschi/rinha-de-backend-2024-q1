## Ferramentas necessárias

- Nodejs
- Docker
- Docker-Compose

## Tecnologias usadas pelo projeto

- Fastify para servidor http
- Pg para conexão com o banco de dados postgres
- Pino para logs

## Como executar

Na raiz do projeto você pode rodar o comando

- docker-compose up
